Bally AS-2518-51 Soundboard for the Tang Nano 9K Dev FPGA Board.

Notes:
Controls are PS2 keyboard Keys.
Consult the Schematics Folder for Information regarding peripheral connections.

Build:
* Obtain Bally U4 Sound roms, see make sound prom script in tools folder for rom filenames.
* Unzip rom files to tools folder.
* Run the make sound proms script in the tools folder.
* Place the generated prom files inside the proms folder.
* Open the AS-2518-51_Snd project file using Gowin and compile.
* Program Tang Nano 9K Board.
